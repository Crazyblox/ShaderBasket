# 🧺 ShaderBasket
A lightweight HTML wrapper for loading, editing, and viewing fragment ShaderToy shaders locally.

## Features
- **Local Loading:** Load .glsl files directly from your machine.
- **Basic Editor:** Built-in interface for real-time code editing.
- **Time Control:** Adjustable time uniform and Play/Pause functionality.
- **Responsive:** WebGL canvas scales automatically with window size.

## Usage
- Open shaderbasket.html in any modern web browser.
- Make changes to the shader display with options at the top left.
- Improve local usage by downloading the .html and/or converting into a web app.
